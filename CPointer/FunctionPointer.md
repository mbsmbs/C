# 함수 포인터
- 어떤 함수를 실행해주려면 그 함수의 시작주소를 변수에 기억하고 있어야 한다.

## 선언
- 반환형 (*변수명) (매개변수 목록); 

- 함수의 시작 주소를 저장하는 변수
- 매개변수 목록과 반환형을 반드시 표기해야함

## 읽는 방법
- righr-left rule
```c
double (*func)(double, double)
   4    2  1          3
```
1. 변수의 이름은 func
2. 이 변수는 포인터를 받아요
3. 2개의 double형을 매개변수로 받고
4. double형을 반환한다

## 함수 포인터 배열
```c
int (*ops[])(int, int) = {add, sub, mul, div};
```

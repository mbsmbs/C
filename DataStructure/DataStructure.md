# 자료구조 기초

## 1. 배열
- 여러 자료들을 메모리 덩어리 안에 줄줄이 세워놓은 구조
- 색인으로 각 요소에 접근

### 배열의 삽입 O(n)
- 끝에 넣는 경우
- 끝이 아닌 경우 : 삽입하려는 위치의 요소부터 마지막 요소를 모두 뒤로 밀고 삽입

### 배열의 삭제 O(n)
- 삭제하는 색인을 기준으로 그 뒤의 값들을 한 칸씩 모두 앞으로

### 배열의 검색 O9n)
- 배열의 요소들을 처음부터 차례대로 방문하여 확인
  - 성공 : 해당 색인 반환
  - 실패 : -1 반환
  
### 배열의 접근 O(1)
- 이미 색인을 알고 있다면 곧바로 접근
